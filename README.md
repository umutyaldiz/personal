# Umut Yaldız - AI-Authored Portfolio

This website is a unique experiment: it is **100% developed, designed, and maintained by autonomous AI agents**.

## 🤖 AI-Driven Development

Every line of code, every architectural decision, and every commit in this repository has been generated by AI. This project serves as a living demonstration of the capabilities of modern Large Language Models (LLMs) and AI coding assistants in building production-ready web applications.

### Key Features:
- **Autonomous Scaffolding:** The project structure was entirely determined and implemented by AI.
- **Zero Manual Human Coding:** No human developer has manually written or edited the code for this site.
- **AI-Managed Lifecycle:** All updates, feature additions, and maintenance tasks are performed by AI agents based on high-level prompts.

## Tech Stack
- **Vite** - Modern build tool
- **Tailwind CSS v4** - Styling engine
- **HTML5** - Semantic structure

## Vision
The goal of this project is to explore the boundaries of "AI-native" software development and to demonstrate how professional-grade web experiences can be built and iterated upon without manual human coding.
